API

Activate Task allows users to mark a task as active or in progress. It is a way to indicate that a task has been started and is currently being worked on.
Terminate Task allows users to end a task that is no longer required or cannot be completed, by either deleting the task or marking it as terminated/cancelled.
Chain Task allows users to link multiple tasks together to form a sequence of tasks that need to be completed in a specific order.
By end the running task and active another one.
Schedule If a higher-priority task is ready, the running task goes back to ready state, its priority is reset, context is saved, and the higher-priority task is executed. Otherwise, the calling task is continued
Get Task ID Return the ID of the running task.
Get Task State Return the state of specific task.
